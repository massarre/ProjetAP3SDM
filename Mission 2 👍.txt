# Liaison SWT-16 (Principal) vers SWT-04
Configuration du mode trunk  
  interface FastEthernet 2/0/1
  switchport mode trunk
  switchport trunk allowed vlan 100,200,300,400

 Configuration du protocole STP (Spanning Tree Protocole)
spanning-tree mode rapid-pvst
spanning-tree vlan 1,100,200,300,400 priority 0
Désactivation de STP sur les ports vers les terminaux (pas nécessaire ! seul shutdown suffit) pareil pour tous les commutateurs
 interface range FastEthernet . . . .
 spanning-tree portfast
 spanning-tree bpduguard enable
